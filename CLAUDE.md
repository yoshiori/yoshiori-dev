# yoshiori.dev

Yoshiori Shoji's personal resume site.

## Stack
- Astro 6 (static build) + Tailwind CSS 4 (`@tailwindcss/vite`)
- Node.js 24 (pinned via `.node-version`, local dev uses mise with `mise.toml`)
- Hosted on Cloudflare Pages (yoshiori.dev)
- SpeakerDeck talks synced weekly via GitHub Actions

## Commands
- `npm run dev` — dev server at localhost:4321
- `npm run build` — build to dist/
- `python scripts/fetch_speakerdeck.py` — fetch latest talks from SpeakerDeck RSS
- `python scripts/generate_og_image.py` — regenerate OG image (requires Pillow)

## Structure
- `src/pages/index.astro` — main page
- `src/components/` — reusable components (ExperienceItem, TalkCard, SectionHeader)
- `src/styles/global.css` — Tailwind `@theme` tokens and base styles
- `src/content/talks.json` — auto-generated by fetch script, do not edit manually
- `src/content/awards.json` — recognition/awards data
- `scripts/fetch_speakerdeck.py` — SpeakerDeck RSS fetcher
- `scripts/generate_og_image.py` — OG image generator (Pillow)
- `.github/workflows/fetch-speakerdeck.yml` — weekly sync workflow

## Design
Dark theme, electric yellow (#e8ff00) accent.
Bebas Neue (display) + IBM Plex Mono + IBM Plex Sans JP.
Design tokens defined in `src/styles/global.css` via Tailwind `@theme`.
All styling uses Tailwind utility classes.
